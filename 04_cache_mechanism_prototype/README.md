# Cache Mechanism Prototype

The cache mechanism is planned but not fully implemented yet.

## Planned Goal

The underwater sensor and edge node are treated as colocated. The unstable part is the submarine cable link between the edge node and the onshore server.

The cache mechanism will evaluate:

- How edge cache size affects data loss rate.
- How dynamic link degradation affects queue buildup.
- How much cache is required to reduce data loss under poor link conditions.

## Current Status

This directory intentionally contains only the current status and implementation plan. No incomplete prototype is presented as finished work.
